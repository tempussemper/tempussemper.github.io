Практическая работа №5

По примеру макета страницы с набором стилей CSS создайте два блока в области "Редактирование HTML кода" navigation и content.

Задайте стили для этих блоков, чтобы они органично вписывались в макет страницы.

Измените стили для "чердака", "подвала" и левой панели страницы.
